### 專案簡介

高中生點日記是以班級為單位的研究計畫，旨在了解高中生同儕之間的互動狀況。

參與者須以「**接觸**」（一對一或一對多皆可）為單位進行記錄，並填答例行性日記問題。

網站將展示個人完成狀況，供用戶進一步查看、篩選與修改，並以點數兌現執行月度結算。
